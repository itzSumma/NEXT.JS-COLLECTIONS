# 🚀 My StartKit Dashboard Project

This is a simple full-stack dashboard project I built while learning **Next.js (App Router)**, authentication, and modern UI design.

---

## 📌 Project Overview

StartKit is a beginner-friendly dashboard web app. It has login/signup, a dashboard layout, and some basic pages like books and home.

I made this project to practice real-world frontend development using Next.js.

---

## ✨ Features

### 🔐 Authentication

- Email & password signup/login
- User session system

### 📊 Dashboard

- Sidebar navigation
- Simple dashboard layout
- Multiple pages support

### 📚 Book UI

- Book cards display
- Clean grid layout

### 🌗 Dark Mode

- Light and dark theme support using next-themes

### 🎨 UI Design

- HeroUI components
- Gravity UI icons
- Toast messages
- Simple hover effects

### 📱 Responsive

- Works on mobile, tablet, and desktop

---

## 🧰 Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- HeroUI
- Next Themes
- MongoDB (for future backend use)
- Better Auth
- React Toastify

---

## 📁 Project Structure

```text
├── app/              # Next.js App Router (Layouts, Pages, Components)
├── public/           # Static assets (images, fonts, icons)
├── AGENTS.md         # Instructions for AI agents (Next.js rules)
├── CLAUDE.md         # Claude-specific configuration
├── next.config.js    # Next.js configuration settings
├── package.json      # Project metadata and dependencies
├── tsconfig.json     # TypeScript configuration
└── .env              # Environment variables (Authentication & DB keys)
```
